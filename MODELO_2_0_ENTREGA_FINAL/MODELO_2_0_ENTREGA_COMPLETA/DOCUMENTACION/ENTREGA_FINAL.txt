🚀 OPCIONES DE INSTALACIÓN
═══════════════════════════════════════════════════════════════════════════════

OPCIÓN A: WINDOWS (1 CLICK)
  1. Doble-click en: SETUP_WINDOWS.bat
  2. Ejecutar: streamlit run streamlit_app.py
  ⏱️ Tiempo: 5 minutos

OPCIÓN B: LINUX/macOS (3 COMANDOS)
  1. chmod +x SETUP_LINUX_MAC.sh
  2. ./SETUP_LINUX_MAC.sh
  3. source venv/bin/activate && streamlit run streamlit_app.py
  ⏱️ Tiempo: 5 minutos

OPCIÓN C: DOCKER (1 COMANDO)
  1. docker-compose up
  ⏱️ Tiempo: 2 minutos

OPCIÓN D: SERVIDOR (COMPLETO CON NGINX + SSL)
  Ver INSTALL.md para instalación en servidor de producción
  ✅ Incluye systemd service, nginx reverse proxy, SSL/HTTPS
