🚀 COMIENZA AQUÍ EN 3 PASOS

PASO 1: Lee esto primero
   → Abre: APLICACION/modelo-2-0-energia/QUICK_START.md

PASO 2: Instala (elige una opción)
   
   Opción A - Windows (1 click):
     Doble-click en: APLICACION/modelo-2-0-energia/SETUP_WINDOWS.bat
   
   Opción B - Linux/macOS:
     chmod +x APLICACION/modelo-2-0-energia/SETUP_LINUX_MAC.sh
     ./APLICACION/modelo-2-0-energia/SETUP_LINUX_MAC.sh
   
   Opción C - Docker:
     cd APLICACION/modelo-2-0-energia
     docker-compose up

PASO 3: Ejecuta
   streamlit run streamlit_app.py
   
   Acceso: http://localhost:8501
